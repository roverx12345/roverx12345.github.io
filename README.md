# roverx12345.github.io

Personal homepage powered by GitHub Pages.

## Edit Profile

Update site title, author information, links, and email in `_config.yml`.

## Edit Publications

Add or update items in `_data/publications.yml`.

## Write Blog Posts

Create Markdown files in `_posts/`:

```text
YYYY-MM-DD-post-title.md
```

Each post should start with front matter:

```yaml
---
title: "Post Title"
description: "One sentence summary."
---
```

## Local Preview

If Ruby and Bundler are installed:

```bash
bundle exec jekyll serve
```

The site also works on GitHub Pages directly after pushing to `main`.
